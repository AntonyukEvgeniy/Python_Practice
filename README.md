## Тестирование
В проекте используется pytest. 

Основные функции покрыты тестами с параметрами. Для генерации отчета в html запустите команду:
```sh
pytest --cov=src --cov-report=html
```
Запуск тестов:
```sh
pytest tests
pytest -k test_name
```
